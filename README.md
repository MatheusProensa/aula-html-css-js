# Exercício HTML, CSS e JavaScript

Projeto desenvolvido como prática de lógica de programação e desenvolvimento web utilizando HTML, CSS e JavaScript na disciplina de Sistemas de Informação.

## Tecnologias utilizadas
HTML  
CSS  
JavaScript  

## Objetivo
Aplicar conceitos básicos de desenvolvimento web e lógica de programação, incluindo manipulação de dados, estruturas condicionais e repetição.

## Exercícios desenvolvidos

O projeto é composto por diferentes atividades separadas em pastas, cada uma com uma funcionalidade específica:

- Verificação de números ímpares
- Soma de valores até 100
- Exibição de nome e idade
- Soma de valores informados pelo usuário
- Tabuada
- Geração de número aleatório
- Outros exercícios de lógica básica

## Estrutura do projeto

O projeto foi organizado em pastas separadas para cada exercício, facilitando o estudo e a prática individual de cada conceito.
